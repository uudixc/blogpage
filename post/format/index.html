format实验

formatted_string = "string {} string {}".format(value1, value2)

其中，花括号 {} 表示需要填充的位置，可以通过调用 format() 函数中的参数来进行填充。如上述代码中，第一个 {} 被 value1 所填充，第二个 {} 被 value2 所填充。

还可以在花括号中指定参数的格式。例如，{:.2f} 表示将浮点数保留两位小数输出。
下面是一些常用的格式化方法：

+ {:d}：整数 
+ {:f}：浮点数
+ {:s}：字符串
+ {:x}：十六进制整数
+ {:o}：八进制整数
+ {:b}：二进制整数

例如

num = 123
string = "The number is {:d}".format(num)
print(string)

#输出为
The number is 123

num = 123.66666
string = "The number is {:f}".format(num)
print(string)

#输出为
The number is 123.666660

num = 123.66666
string = "The number is {:.3f}".format(num)
print(string)

#输出为
The number is 123.666

num = '123.66666'
string = "The number is {:s}".format(num)
print(string)

#输出为
The number is 123.66666

还可以使用位置参数和关键字参数。

例如：

string = "我的名字是 {0}, 年龄为 {1}, 来自 {country}".format("Tom", 25, country="China")
print(string)

#输出为
我的名字是Tom,年龄为25，来自China

除了使用位置参数和关键字参数，还可以使用字典或对象的属性。

例如：

person = {'name': 'Tom', 'age': 25,'country': 'China'}
string = "我的名字是 {person[name]}, 年龄为 {person[age]}, 来自 {person[country]}".format(person=person)
print(string)

#输出为
我的名字是Tom,年龄为25，来自China

格式控制标记

格式控制标记包括 :<填充><对齐><宽度><,><精度><类型>
format() 函数可以使用对齐和填充字符来调整输出的格式。
例如：

num = 123
string = "{:<10d}".format(num)
print(string)

#输出为：
123


num = 123
string = "{:>10d}".format(num)
print(string)

#输出为：
      123


num = 123
string = "{:^10d}".format(num)
print(string)

#输出为：
  123

其中，< 表示左对齐，> 表示右对齐，^ 表示居中对齐 10表示输出宽度为 10，多余的部分使用空格填充
多余部分还可以使用其他字符填充，例如：

num = 123
string = "{:-^10d}".format(num)
print(string)

#输出为：
---123----

<类型>表示输出整数和浮点数的格式规则
整数

  • b: 输出整数的二进制方式
  • c: 输出整数对应的Unicode字符
  • d: 输出整数的十进制方式
  • o: 输出整数的八进制方式
  • x: 输出整数的小写十六进制方式
  • X: 输出整数的大写十六进制方式

例如

print("{0:b},{0:c},{0:d},{0:o},{0:x},{0:X}".format(666))

#输出为
1010011010,ʚ,666,1232,29a,29A

浮点数

  • e: 输出浮点数对应的小写字母e的指数形式
  • E: 输出浮点数对应的大写字母E的指数形式
  • f: 输出浮点数的标准浮点数形式
  • %: 输出浮点数的百分形式

例如

print("{0:e},{0:E},{0:f},{0:%}".foramt(7.5692))

#输出为
7.569200e+00,7.569200E+00,7.569200,756.920000%

使用<精度>控制输出精度
例如

print("{0:.3e},{0:.3E},{0:.3f},{0:.3%}".foramt(7.5692))

#输出为
7.569e+00,7.569E+00,7.569,756.920%
